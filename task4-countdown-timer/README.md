# Countdown Timer — NovaStack Hub Internship (Task 4)

A responsive countdown timer app with flip-card animations built with vanilla HTML, CSS, and JavaScript.

## Features
- User picks a target date & time via a datetime-local input
- Live countdown displaying Days, Hours, Minutes, Seconds
- Flip-card style digit animation when values change
- "Time's Up!" celebration screen when countdown hits zero
- Reset / set new timer functionality
- Fully responsive

## Tech
- HTML5
- CSS3 (3D transforms, perspective, animations)
- Vanilla JavaScript (ES6+, setInterval)

## How to Run
Open `index.html` in any modern browser. No build tools required.
