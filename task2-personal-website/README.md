# Personal Portfolio — NovaStack Hub Internship (Task 2)

A modern, one-page personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features
- Sticky navbar with smooth scroll and active section highlighting
- Hero section with gradient-text name and dual CTAs
- About Me with photo placeholder and details
- Skills grid with hover-animated cards
- 3 project cards with tag badges and live demo links
- Contact form with basic validation
- Responsive design (mobile, tablet, desktop)
- Dark theme with consistent spacing and typography

## Tech
- HTML5
- CSS3 (Flexbox, Grid, custom properties, backdrop-filter)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## How to Run
Open `index.html` in any modern browser. No build tools required.

## Personalization
Replace all `[Placeholder]` values:
- `[Your Name]` — your full name
- `[Your Title]` — your professional title
- `[Your short tagline...]` — a brief intro
- `[Your Bio...]` — about-me paragraph(s)
- `[Your Photo]` — your profile image URL
- `[Your City, Country]` and `[your.email@example.com]`
- 3 project placeholders with your real projects
- Social links in the footer are empty `href="#"` — add your profile URLs
