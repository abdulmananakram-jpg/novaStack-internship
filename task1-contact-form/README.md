# Contact Form — NovaStack Hub Internship (Task 1)

A professional, responsive contact form built with vanilla HTML, CSS, and JavaScript.

## Features
- Client-side validation with real-time error messages
- Simulated submission with loading spinner and success animation
- Card-style layout with gradient background
- Fully responsive (mobile, tablet, desktop)
- Accessible markup, semantic HTML5, proper labels

## Tech
- HTML5
- CSS3 (Flexbox, Grid, custom properties, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## How to Run
Open `index.html` in any modern browser. No build tools or server required.

## Notes
- Form uses `mailto:` as a placeholder (no backend). Replace `action` in the `<form>` tag if you connect a real backend.
